# Projeto Atividade Extensionista II - UNINTER
## 📍 Visão Geral
Este projeto foi desenvolvido como parte da Atividade Extensionista II do Centro Universitário Internacional UNINTER. O objetivo central é combater a exclusão informativa em comunidades escolares de difícil acesso, facilitando o fluxo de comunicados, calendários e documentos essenciais entre a instituição e os responsáveis.

## ⚠️ O Problema
Em muitas regiões, o acesso à internet é instável ou limitado, e o deslocamento físico até a escola é dificultado por fatores geográficos ou de infraestrutura. Isso resulta em:

Pais desinformados sobre reuniões e eventos.

Perda de prazos para matrículas e entrega de documentos.

Dificuldade no acompanhamento da vida escolar do aluno.

## 🚀 A Solução
O foco deste projeto é criar uma ponte tecnológica de baixa barreira, utilizando ferramentas que consomem poucos dados e funcionam em dispositivos modestos, garantindo que a informação chegue à ponta final de forma clara e organizada.

Principais Funcionalidades Propostas:
Central de Comunicados Offline-First: Interface leve para visualização de avisos pré-carregados.

Notificações de Baixo Consumo: Integração com SMS ou disparos via APIs de mensagens instantâneas (ex: WhatsApp/Telegram) para alertas urgentes.

Repositório de Documentos: Acesso facilitado a listas de materiais, cardápios escolares e calendários de provas.

Painel Administrativo Simplificado: Interface para que a secretaria, diretores e professores da escola possam gerenciar os avisos sem necessidade de conhecimentos técnicos avançados.

## 🛠️ Tecnologias
Para garantir a acessibilidade, o projeto prioriza:

Frontend: ReactJS (foco em performance).

Mobile: Progressive Web App (PWA) — permite que o app seja "instalado" no celular sem ocupar muito espaço e funcione parcialmente offline. (será construido)

Backend: C#(.NET)

Dashboard: Python (Em construção)

Banco de Dados: Postgres
